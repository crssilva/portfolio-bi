# Dashboard de Vendas - Power BI

## 🎯 Objetivo do Projeto
Desenvolver um dashboard interativo em Power BI para análise comercial, permitindo acompanhar indicadores de vendas, lucro e performance dos vendedores. O objetivo é transformar dados em insights visuais para apoiar a tomada de decisão.

## 📊 Principais KPIs Monitorados
- **Faturamento Total**
- **Lucro Total**
- **Quantidade Vendida**
- **Produto Mais Vendido**
- **Top Vendedor (nome e valor)**
- **Distribuição por forma de pagamento**
- **Evolução mensal de vendas e quantidade**

## 🛠️ Tecnologias Utilizadas
- **Power BI** para criação de dashboards interativos
- **DAX** para personalização de medidas e inserção de nomes/fotos
- **Excel** como base de dados
- **Filtros dinâmicos** para análise detalhada

## 📈 Funcionalidades
- Filtros para produto mais vendido e top vendedor
- Gráficos de faturamento e quantidade vendida por mês
- Relatórios de vendas por vendedor
- Distribuição de vendas por forma de pagamento
- Personalização visual com fotos e nomes via DAX
